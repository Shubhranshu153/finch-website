# Uninstalling Finch

The best way to uninstall Finch on Windows is by using the standard method via the Settings app. 

## Uninstalling Finch via Settings app

1. Open the Settings app and navigate to the "Apps & features" subpage
1. Search for "Finch"
1. Click "Uninstall"
1. Done!
